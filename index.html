Spotify Playlist Importer

1
Connecter ton compte Spotify
⚠️ Étape obligatoire avant de te connecter Dans ton Spotify Dashboard → ton app → Edit Settings → User Management
Ajoute ton adresse email Spotify. Sans ça, la création de playlist sera bloquée (erreur 403).

🔗 URI de redirection à copier dans ton app Spotify :
Pas encore de Client ID ? → developer.spotify.com/dashboard → Create App → colle l'URI ci-dessus dans Redirect URIs → coche Web API → Save

✓ Connecté :


2
Charger ton fichier CSV
📂
Glisse ton CSV ici ou clique pour choisir Format attendu : colonnes Artist, Track
3
Nommer et créer la playlist
⏱️ L'import est intentionnellement lent (1 titre/seconde) pour respecter les limites de Spotify. 150 titres ≈ 3 minutes. Ne ferme pas l'onglet.
🎉
Titres trouvés !

Ouvrir la playlist  
📋 Colle ces URIs dans Spotify Desktop
1. Ouvre Spotify Desktop sur ton ordi
2. Ouvre la playlist créée (lien ci-dessus)
3. Clique dans la playlist → Ctrl+V (ou Cmd+V sur Mac)
4. Tous les titres s'ajoutent instantanément ✅
✅ URIs copiées dans le presse-papier !
